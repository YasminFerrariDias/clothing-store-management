# Gestão de Loja de Roupas

Esquema de banco de dados relacional para gerenciar uma loja de roupas, com tabelas de clientes, produtos, fornecedores, vendas, pedidos, estoque e endereços.

## Tecnologias

- [SQL](https://pt.wikipedia.org/wiki/SQL) — modelagem e manipulação do banco de dados

## Sobre

O script inclui a criação das tabelas, inserção de dados iniciais e definição de chaves primárias e estrangeiras, estabelecendo as relações entre as entidades do sistema.
